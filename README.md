# 🎨 E-Commerce Frontend

Фронтенд для интернет-магазина, разработанный на **React**, **TypeScript** и **Vite**.  
Проект включает страницы каталога, корзину, авторизацию и оформление заказа.


## 🚀 Технологии

- React
- TypeScript
- Vite
- React Router
- SCSS / CSS Modules
- Fetch API
